What is HTML?

Hypertext Markup Language (HTML) is the standard language for creating web pages and web
applications.

Web browsers receive HTML documents from a web server of from local storage and render the documents
into multimedia web pages. HTML describes the structure of a web page semantically and originaly
included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. Width HTML constructs, images and other object
such as interactive forms may be embedded into the reneder page.HTML provides a means to create
structured documents by denoting structural semantics for text such as heading, paragraphs,lists,links,
quotes and other items.HTML elements are delineated be togs, written using angle brackets . Tags such as
< img/ > and < input > directly introduce content into the page. Other tags such as < p >surround and
provide information about documents text and may include other tags as sub-elements. Browsers do not
display theHTML tags, but use them interpred the content of the page.

Markup

HTML markup consist of several key components, including those caled logs (and their otributes),
character based data types, character references and entity references. HTML tags most commonly come in
pairs like < h1 > and, < /h1 > althought some represent empty elements and so are unpared for example
< img >. The first tag in such a pair is the start tag, and the second is the end tag they are also called opening
tags and closing tags.

Another important component is the HTML document type declaration, which triggers standarts mode
rendering.

Angular JS. Основы

Подключение

1.скачать скрипт с сайта https://angularjs.org/
2. указать ссылку на скрипт "https://ajax.googleapis.com/ajax/libs/angularjs/1.8.2/angular.min.js"

В теле документа поместить тег "скрипт" и в разметке указать область действия приложения Angular (ng-app). Чаще всего это максимальная область: <html ng-app>

Внутри области действия приложения будут обрабатываться выражения .Выражения заключаются { { 2+3 } } {{ 2+3 }}

Использование
